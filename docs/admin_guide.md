# Admin User Guide

## Accessing the Dashboard
1. Navigate to `/admin`.
2. Log in with your credentials (default: `admin@example.com` / `admin123`).

## Managing Reports
### Generating a Report
1. Go to the **Reports** page.
2. Click **Generate Report**.
3. Select the sport (e.g., NFL, NBA).
4. Click **Generate**. The system will fetch data and create a draft report.

### Editing Picks
1. In the report view, click **Edit** on a pick card.
2. Update the **Units**, **Hierarchy** (Lock, Featured, etc.), and **Analysis**.
3. Click **Save**.

### Publishing
1. Once all picks are reviewed, click **Publish Report**.
2. Confirm the action. The report is now live on the client portal.

## Managing Player Props
1. Expand a pick to see details.
2. Use the **Player Props** section to add or remove props associated with the game.
